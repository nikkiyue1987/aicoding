# 英语课文跟读工具 - 项目概述

## 项目结构

### 页面架构
1. **设置页面 (index.html)**
   - 标题区：英语文章跟读
   - 短文输入区：多行文本输入
   - 短文释义输入区：中文释义输入
   - 单词详情输入区：单词、音标、词性、释义格式
   - 图片上传区：场景插图上传
   - 参数设置区：字号、语速、音色、停顿时长
   - 开始按钮：跳转到跟读页面

2. **跟读页面 (reading.html)**
   - 顶部标题区：学习目标、短文主题
   - 词汇统计区：单词总数、词性分布
   - 插图区：场景图片显示
   - 单词讲解区：分色块词汇列表
   - 短文正文区：高亮文本、朗读进度
   - 控制区：播放/暂停、中英切换

## 核心功能模块

### 数据管理
- 本地存储用户输入的短文、释义、单词详情
- 参数设置的保存和读取
- 图片文件的本地存储

### 语音功能
- 使用Web Speech API进行文本转语音
- 支持多种音色选择
- 语速调节和停顿控制
- 朗读进度可视化

### 词汇处理
- 自动分句功能
- 词汇分级（高频、中频、低频）
- 词性统计和分类
- 颜色编码系统

### 用户界面
- 响应式设计适配不同设备
- 现代化视觉风格
- 平滑的动画效果
- 直观的交互反馈

## 技术栈
- HTML5 + CSS3 + JavaScript
- Tailwind CSS框架
- Web Speech API
- Local Storage API
- Canvas API（图片处理）
